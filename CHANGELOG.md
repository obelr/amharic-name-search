# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-28

### Added
- Initial release
- `transliterateToAmharic()` function for converting English names to Amharic
- `matchesName()` function for matching names across scripts
- `expandSearchQuery()` function for expanding search queries
- Common name mappings for 50+ Ethiopian names
- Full TypeScript support with type definitions
- Comprehensive test suite
- Documentation and usage examples
